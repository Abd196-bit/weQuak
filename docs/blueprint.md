# **App Name**: QuackChat

## Core Features:

- User Registration: Allow users to register with a username and password, storing hashed passwords in Firebase.
- User Login: Authenticate users against stored credentials in Firebase.
- Room Listing: Display a list of available chat rooms fetched from Firebase.
- Message Sending: Allow users to send text messages to a selected room, which are stored in Firebase.
- Message Loading: Load and display messages for a specific room from Firebase.
- Real-time Chat Updates: Integrate Firebase real-time database to provide real-time updates to the chat UI when new messages are sent.

## Style Guidelines:

- Primary color: Yellow (#FFDA63) for fun and energy.
- Background color: Black (#000000) for a modern, high-contrast look.
- Accent color: Bright yellow (#FFFF00) to provide contrast for interactive elements and notifications.
- Body and headline font: 'PT Sans' for readability and a modern feel.
- Use minimalist icons to represent different chat functions (send, attach, settings).
- Clean and intuitive message display with clear distinction between sent and received messages.
- Subtle animations for new messages and user interactions to enhance the chat experience.
